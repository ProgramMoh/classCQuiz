# CA DMV Class C Practice Test

Personal study tool for preparing for the California Class C driver license knowledge test.

## Features

- **Quiz Mode** — Practice with 10, 20, 30, or 46 questions (DMV simulation). Filter by category. Shows correct answers and explanations after each question.
- **Study Mode** — Browse all 150+ questions with expandable answer cards. Search and filter by category.
- **Tips** — Key differences between UAE and California driving, DMV test day tips, and important numbers to memorize.
- **Score History** — Track your quiz results during the session.

## Tech Stack

Pure HTML/CSS/JavaScript — no build tools, no frameworks, no dependencies.

## Development

Open `index.html` in a browser, or serve with any static file server:

```bash
# Python
python3 -m http.server 8000

# Node
npx serve .
```

## Deployment

This is a static site — deploy to Vercel, GitHub Pages, Netlify, or any static host.

### Vercel
```bash
npm i -g vercel
vercel
```

### GitHub Pages
Push to a GitHub repo and enable Pages in Settings → Pages → Source: main branch.
